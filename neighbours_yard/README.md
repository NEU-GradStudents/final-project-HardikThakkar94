# Neighbours_Yard

A new Flutter project.

## Pre-requisties

1. Intall Java.
2. Install Android Studio
3. Install Flutter.

## Getting Started
This project is a starting point for a Flutter application. (Clone and open this repo in your IDE i.e, Android Studio)

```bash
$ flutter clean
$ flutter upgrade
$ flutter pub get
$ flutter run
```
